# ⚡ 博客 CMS - 快速启动指南

## 🎯 三种最快启动方式

---

### 方式一：Launchpad（推荐新手）

```
1. 按 F4 键
   或四指向内捏合触控板

2. 输入「博客」或「CMS」

3. 点击带有蓝紫色笔记本图标的「博客CMS」

4. 等待浏览器自动打开（约3秒）
```

**优点：** 直观、简单、无需记住位置

---

### 方式二：Spotlight（推荐高手）

```
1. 按 ⌘ + 空格

2. 输入「博客」

3. 看到「博客CMS」后按回车

4. 等待浏览器自动打开
```

**优点：** 最快捷、不离开键盘

---

### 方式三：Dock 栏（日常使用）

**首次设置：**
```
1. 打开「访达」(⌘ + N)

2. 点击左侧「应用程序」(⌘ + Shift + A)

3. 找到「博客CMS」(蓝紫色笔记本图标)

4. 拖动到 Dock 栏底部

5. 松手完成
```

**日常使用：**
```
点击 Dock 栏上的博客CMS图标 → 完成！
```

**优点：** 一键启动、最方便

## 🚦 首次启动注意事项

### macOS 安全提示
第一次启动时，可能会看到：

```
「博客CMS」来自身份不明的开发者，无法打开。
```

**解决方法：**

**方法一（推荐）：**
1. 打开「系统偏好设置」
2. 点击「安全性与隐私」
3. 在底部看到「"博客CMS"已被阻止」
4. 点击「仍要打开」
5. 再次双击应用

**方法二：**
1. 按住 Control 键
2. 点击应用图标
3. 选择「打开」
4. 点击「打开」确认

完成后，以后启动就不会再有提示了。

## 💡 使用技巧

### 技巧1：添加到 Dock
- 建议放在 Dock 最右侧（文档左边）
- 方便日常快速启动

### 技巧2：键盘启动
1. ⌘ + 空格 打开 Spotlight
2. 输入「博」第一个字
3. 回车启动
4. **总共 2 秒完成！**

### 技巧3：文件夹快捷方式
- 在桌面创建应用程序的替身：
  1. 找到 `/Applications/博客CMS.app`
  2. 按 ⌘ + L 创建替身
  3. 拖到桌面

---

## 🛑 停止服务器

使用完毕后：

**方法一：关闭 Terminal**
- 直接关闭 Terminal 窗口（⌘ + W）
- 服务器自动停止

**方法二：快捷键**
- 在 Terminal 中按 Ctrl + C
- 服务器立即停止
