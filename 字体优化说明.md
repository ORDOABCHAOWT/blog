# 🎨 编辑器字体优化详解

## 📖 字体选择理念

### LXGW WenKai (霞鹜文楷) - 主字体

**为什么选择文楷？**

1. **诗意美学**
   - 融合楷体的端庄与行书的灵动
   - 笔画舒展自然，富有韵律感
   - 特别适合文学创作和长文写作

2. **技术优势**
   - 开源免费，商用无忧
   - 完整支持 GB2312、GB/T 12345 字符集
   - 字形规范，符合现代阅读习惯

3. **中英文混排**
   - 中英文字符高度协调
   - 标点符号位置恰当
   - 数字和字母清晰易读

## 📐 排版参数详解

### 对比表格

| 参数 | 优化前 | 优化后 | 提升效果 |
|------|--------|--------|----------|
| 字号 | 14px | 16px | ⬆️ 14% |
| 行高 | 1.6 | 2.0 | ⬆️ 25% |
| 字间距 | 0 | 0.5px | ⬆️ 新增 |
| 段落内边距 | 0 | 16px | ⬆️ 新增 |
| 行内边距 | 0 | 2px | ⬆️ 新增 |

### 字号：14px → 16px

**原因：**
- 14px 对长时间阅读偏小
- 16px 是舒适阅读的黄金字号
- 在 Retina 屏幕上显示更清晰

**效果：**
- ✅ 减轻视觉疲劳
- ✅ 提升文字辨识度
- ✅ 增强专注力

### 行高：1.6 → 2.0

**原因：**
- 1.6 的行距对中文偏紧
- 2.0 符合传统中文排版美学
- 接近古籍版式的行距比例

**效果：**
- ✅ 视线移动更流畅
- ✅ 避免视觉拥挤感
- ✅ 提升诗意氛围

### 字间距：0 → 0.5px

**原因：**
- 中文字符适当留白更优雅
- 增强文字呼吸感
- 模拟传统活字印刷效果

**效果：**
- ✅ 字符边界清晰
- ✅ 减少视觉疲劳
- ✅ 提升文学气质

## 🌟 字体降级方案

```css
font-family:
  'LXGW WenKai',           /* 首选：霞鹜文楷 */
  'Songti SC',             /* 备选：macOS 宋体 */
  'Noto Serif SC',         /* 备选：思源宋体 */
  'Source Han Serif SC',   /* 备选：思源宋体 */
  'STSong',                /* 备选：华文宋体 */
  'SimSun',                /* 备选：中易宋体 */
  'Georgia',               /* 英文衬线字体 */
  'Times New Roman',       /* 英文衬线字体 */
  serif;                   /* 系统默认 */
```

**降级策略：**
1. 优先使用 LXGW WenKai（通过 CDN 加载）
2. 若加载失败，使用系统内置宋体
3. 保证在任何情况下都有优雅的阅读体验

## 💻 技术实现

### CDN 加载
在 `layout.tsx` 中引入：
```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/lxgw-wenkai-webfont@1.1.0/style.css"
/>
```

### CSS 样式
在 `MarkdownEditor.tsx` 中应用：
```jsx
<style jsx global>{`
  .markdown-editor .CodeMirror {
    font-family: 'LXGW WenKai', ...;
    font-size: 16px;
    line-height: 2;
    letter-spacing: 0.5px;
  }
`}</style>
```
