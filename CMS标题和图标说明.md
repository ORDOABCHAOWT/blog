# 🏷️ CMS 标题和图标区分说明

## ✅ 已完成的优化

CMS 管理后台现在拥有独立的标题和图标，与博客前台完全区分。

---

## 🎯 优化内容

### 1. 独立的浏览器标签标题

**博客前台：**
- 首页：`Taffy's Blog`
- 文章页：`文章标题 | Taffy's Blog`

**CMS 后台：**
- 管理首页：`博客管理后台 | Taffy CMS`
- 写新文章：`写新文章 | 博客管理后台`
- 编辑文章：`编辑：文章标题 | 博客管理后台`

### 2. 独立的 Favicon 图标

**博客前台：**
- 使用默认的博客 favicon

**CMS 后台：**
- 使用深蓝色齿轮图标 ⚙️
- 设计理念：齿轮代表管理、设置、后台操作
- 颜色：深蓝色 (#1E3A8A) + 白色齿轮

---

## 🎨 图标设计

### CMS Favicon 特点

```
设计元素：
- 圆形背景：深蓝色
- 中心图案：白色齿轮（6齿）
- 边框：蓝色高亮
- 中心孔：白色圆形
```

**视觉效果：**
- 深蓝色代表专业、管理
- 齿轮代表系统、配置、后台
- 与博客前台形成清晰区分

## 🔧 技术实现

### 1. 每个 Admin 路由都有独立 Layout

```typescript
// src/app/admin/layout.tsx
export const metadata: Metadata = {
  title: '博客管理后台 | Taffy CMS',
  icons: {
    icon: '/admin-favicon.ico',
  },
};
```

### 2. 动态标题更新

```typescript
// 在编辑页面显示具体文章标题
useEffect(() => {
  if (formData.title) {
    document.title = `编辑：${formData.title} | 博客管理后台`;
  }
}, [formData.title]);
```

### 3. 独立 Favicon

```
public/
├── favicon.ico          # 博客前台图标
└── admin-favicon.ico    # CMS 后台图标 ⚙️
```

## 💡 使用场景

### 场景1：多标签管理
同时打开多个标签时，可以快速识别：
- 蓝色齿轮 = 后台管理
- 默认图标 = 前台博客

书签栏里可以清晰区分：

## 🔍 FAQ

### Q: 为什么要区分标题和图标？
**A:** 当你同时打开博客前台预览和后台编辑时，可以快速区分标签。

### Q: 可以自定义 CMS 图标吗？
**A:** 可以，替换 `public/admin-favicon.ico` 文件即可。

### Q: 图标在哪些地方显示？
**A:**
- 浏览器标签页
- 书签栏
- 历史记录
- 浏览器窗口标题
