## 🚀 启动方式

### 方式一：Launchpad（推荐）
1. 按 F4 或四指捏合触控板打开 Launchpad
2. 搜索或找到「博客CMS」
3. 点击图标启动

### 方式二：访达
1. 打开「访达」
2. 左侧点击「应用程序」
3. 双击「博客CMS」

### 方式三：Spotlight
1. 按 ⌘ + 空格
2. 输入「博客」或「cms」
3. 选择「博客CMS」并回车

### 方式四：添加到 Dock 栏（最快捷）
1. 打开「应用程序」文件夹
2. 找到「博客CMS.app」
3. 拖动到 Dock 栏
4. 以后点击 Dock 图标即可启动

## 📋 使用流程

### 首次启动
1. 双击应用图标
2. 如果出现「无法打开应用」提示：
   - 打开「系统偏好设置」> 「安全性与隐私」
   - 点击「仍要打开」
   - 再次双击应用图标

3. 应用会自动：
   - 打开新的 Terminal 窗口
   - 执行 `npm run dev` 启动服务器
   - 等待 3 秒让服务器启动
   - 自动打开浏览器访问管理界面

### 日常使用
1. 从 Dock 或 Launchpad 点击图标启动
2. 等待浏览器自动打开（约 3-5 秒）
3. 开始写作和管理文章
4. 使用完毕后关闭 Terminal 窗口停止服务器

## 🎯 添加到 Dock 栏教程

**步骤：**

1. **打开访达**
   - 点击 Dock 上的访达图标
   - 或按 ⌘ + N

2. **进入应用程序文件夹**
   - 点击左侧边栏的「应用程序」
   - 或按 ⌘ + Shift + A

3. **找到博客CMS**
   - 向下滚动找到「博客CMS」
   - 图标是蓝紫渐变背景，带笔记本和笔的设计

4. **拖到 Dock**
   - 按住图标拖动到 Dock 栏
   - 放在你喜欢的位置（建议放在最右侧）
   - 松开鼠标

5. **完成**
   - 现在可以直接点击 Dock 图标启动了！

## 🛠️ 故障排除

### 图标没有显示？
1. 尝试重启访达：
   ```bash
   killall Finder
   ```

2. 或者在终端执行：
   ```bash
   touch /Applications/博客CMS.app
   ```

3. 注销并重新登录

### 应用无法启动？
1. 检查权限设置：
   - 系统偏好设置 > 安全性与隐私
   - 允许博客CMS运行

2. 重新赋予执行权限：
   ```bash
   chmod +x /Applications/博客CMS.app/Contents/MacOS/launcher
   ```
