# 博客 CMS 使用指南

## 🚀 快速启动

### 方式一：双击桌面应用（推荐）

1. 在桌面找到「**博客CMS.app**」
2. 双击启动
3. 系统会自动：
   - 在新的 Terminal 窗口启动服务器
   - 打开浏览器访问管理后台
   - 显示通知提示启动成功

### 方式二：使用 .command 文件

1. 双击桌面上的「**博客CMS.command**」
2. 效果与方式一相同

### 方式三：命令行启动（传统方式）

```bash
cd /Users/whitney/Downloads/minimal-blog
npm run dev
```

然后手动打开浏览器访问 `http://localhost:3000/admin`

## ✨ 编辑器字体优化

### 已应用的字体方案

编辑器使用了精心挑选的中英文字体组合：

**主字体：** LXGW WenKai (霞鹜文楷)
- 开源免费的优美中文字体
- 兼顾手写体的诗意与印刷体的规范
- 完美支持中英文混排

**备用字体：**
- Songti SC (宋体) - macOS 系统自带
- Georgia, Times New Roman - 经典英文衬线字体

### 排版优化

- **字号：** 16px（比默认大，更易阅读）
- **行高：** 2（行间距加倍，视觉舒适）
- **字间距：** 0.5px（字符间适当留白）
- **段落间距：** 增加内边距，层次分明

### 视觉效果

- 光标加粗，便于定位
- 标题加粗并增大字间距
- 工具栏浅灰背景，层次清晰
- 圆角边框，现代简洁

## 📝 使用 CMS 管理文章

### 新建文章

1. 点击「✏️ 写新文章」
2. 填写文章信息：
   - **标题**：文章的显示标题
   - **别名 (Slug)**：用于 URL，自动根据标题生成
   - **日期**：发布日期，默认为今天
   - **描述**：文章简介，显示在列表中
3. 在编辑器中写作（支持完整 Markdown 语法）
4. 点击「保存文章」

### 编辑文章

1. 在文章列表中找到要编辑的文章
2. 点击「编辑」按钮
3. 修改内容
4. 点击「保存更改」

### 删除文章

1. 在文章列表中点击「删除」
2. 确认删除

---

## 🚀 一键发布到 GitHub & Vercel

### 发布流程

1. 编辑完文章后，点击「🚀 一键发布」
2. 系统会自动执行：
   ```bash
   git add posts/
   git commit -m "博客更新 - 2025-11-29 11:30:45"
   git push origin main
   ```
3. Vercel 检测到 GitHub 更新后自动部署
4. 大约 1-2 分钟后，你的博客就更新了！

### 注意事项

- 确保本地修改已保存
- 确保有网络连接
- 如果推送失败，检查 Git 配置

## 🎨 Markdown 编辑器功能

### 工具栏功能

- **B**：加粗
- **I**：斜体
- **H**：标题（点击多次切换级别）
- **"**：引用
- **• —**：无序列表 / 有序列表
- **🔗**：插入链接
- **🖼️**：插入图片
- **👁️**：预览模式
- **⚡**：分屏模式（边写边看）
- **⛶**：全屏模式
- **?**：Markdown 指南

### 快捷键

- **⌘ + B**：加粗
- **⌘ + I**：斜体
- **⌘ + K**：插入链接
- **⌘ + ⌥ + I**：插入图片

### Markdown 语法示例

```markdown
# 一级标题
## 二级标题

**加粗文字** *斜体文字*

[链接文字](https://example.com)

![图片描述](图片URL)

> 引用内容

- 列表项 1
- 列表项 2

1. 有序列表 1
2. 有序列表 2
```
