## 🆘 常见问题

### Q: 双击应用没反应？
A: 首次运行需要授权，在「系统偏好设置」>「安全性与隐私」中允许运行

### Q: 字体没有显示为文楷？
A: CDN 加载可能需要时间，刷新页面或检查网络连接

### Q: 一键发布失败？
A: 检查 Git 配置和网络连接，确保有权限推送到 GitHub

### Q: 如何修改启动端口？
A: 编辑 `package.json` 中的 dev 命令，添加 `-p 端口号`
